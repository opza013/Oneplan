# OnePlan — Next Step Save State & Delete Confirmation

Render-ready static package.

## Update
- Activity Next Step Save is disabled and grey when the row is blank or unchanged.
- Save becomes blue and clickable only after valid new data or a real change.
- Delete opens a confirmation modal before removing the Next Step.
- Existing OnePlan features and multi-select/search filters are retained.

## Deploy
Upload `index.html`, `render.yaml`, and `README.md` to the GitHub repository root. Render can then auto-deploy the latest commit.
